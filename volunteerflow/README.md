# VolunteerFlow

A complete volunteer management platform with a professional design matching the Figma mockups.

## 🌟 Features

### Dashboard
- Real-time statistics with interactive cards
- Activity feed showing recent events
- Quick action buttons
- Professional color-coded metrics

### Volunteers Management
- Complete volunteer directory
- Advanced search and filtering
- Status management (Active, Inactive, Pending)
- Skills tracking and hours logging
- Bulk operations

### Event Management
- Create and manage volunteer events
- Category-based organization
- Visual progress indicators
- Real-time participant tracking
- Event status management

### Application Workflow
- Volunteer application submission
- Admin review and approval system
- Bulk approve/reject functionality
- Message tracking
- Status history

## 🎨 Design

- **Font**: Inter (Google Fonts)
- **Primary Color**: #2F80ED (Blue)
- **Success Color**: #27AE60 (Green)
- **Warning Color**: #F29944 (Orange)
- **Danger Color**: #EB5757 (Red)
- **Responsive Design**: Mobile-first approach
